About Razen Web Server

What is Razen?

Razen is a modern programming language with a powerful standard library that includes built-in web server capabilities.

Features

  • Simple and intuitive syntax
  • Built-in HTTP server
  • Fast execution with JIT compilation
  • Rich standard library (11 modules)
  • Easy to learn and use

Server Module

The server module provides HTTP server functionality similar to Go's net/http and Node.js's Express, making it easy to build web applications.

use server

fun main() {
    // Serve files from public/ directory
    server.listen(8080, "./public")
}
